Design system and constraints for XalQberdi infrastructure monitoring app

- Font: Wix Madefor Display (Google Fonts), fallback: system sans-serif — applied globally via * selector
- Primary color: Action Blue #0088FF (HSL 211 100% 50%)
- Destructive/error red: #FF383C (HSL 358 100% 61%)
- Success green: #34C759 (HSL 142 76% 50%)
- Style: mobile-first, rounded corners, light shadows, clean white floating blocks, no gradients/glassmorphism
- No emojis in UI — use lucide-react icons everywhere
- Info cards: show "Bor" / "Yo'q" (no checkmarks/X symbols)
- Language: Uzbek (Latin)
- Toast notifications: success=green bg, error=red bg, white text
