# Фото объектов для карты

Положите сюда изображения (jpg, png, webp). Backend подставит путь в поле `image` по совпадению названия файла (без расширения) с названием объекта.

- **schools/** — школы (объекты с `type: "school"`)
- **universities/** — вузы (`type: "university"`)
- **medical/** — медучреждения (`type: "medical"`)

**Имена файлов** должны совпадать с полем `name` объекта из `data/enriched-objects.json` (можно без № и скобок, регистр не важен).

Примеры:
- `Омегамед Центр.jpg` → объект "Омегамед Центр"
- `Университет Инха.jpg` → "Университет Инха"
- `Школа 42.jpg` или `Школа №42.jpg` → "Школа №42"
- `Клиническая Больница №5 (Мать И Дитя).jpg` → "Клиническая Больница №5 (Мать И Дитя)"

После добавления фото выполните в папке backend:
```bash
npm run build:map
```
И перезапустите сервер (или удалите `data/generated/map-objects.json` и сделайте запрос к API).
